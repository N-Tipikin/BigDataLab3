# Лабораторная работа по дисциплине «Большие данные»

Тема: анализ набора данных **Employee_Salaries** с помощью **Spark SQL** и средств визуализации в среде **Jupyter Notebook**.

Вариант по заданию (№12): датасет Employee_Salaries с сайта  
https://catalog.data.gov/dataset/employee-salaries-2023

## Структура репозитория

- `Employee_Salaries.ipynb` — Jupyter Notebook с решением лабораторной работы:
  - загрузка и подготовка набора данных Boston Housing;
  - перенос данных в Spark DataFrame;
  - выполнение запросов с использованием Spark SQL;
  - 5 упражнений с данными, часть из которых сопровождается визуализациями.
- `Employee_Salaries_-_2023.csv` — текстовый файл с исходным набором данных Boston Housing  
  (сохранённая копия содержимого страницы `https://catalog.data.gov/dataset/employee-salaries-2023`).
- `README.md` — текущий файл с описанием проекта.

> Если вы по какой-то причине не видите `Employee_Salaries_-_2023.csv` в репозитории, его можно скачать вручную по ссылке выше и сохранить в корень проекта под этим именем.

## Содержание лабораторной работы

В рамках лабораторной работы выполняются:

1. Описание и базовый анализ набора данных `Employee Salaries 2023`
   (структура таблицы, типы столбцов, количество наблюдений, базовые статистики).
2. Перенос данных в Spark DataFrame и регистрация временного представления Employee.
3. Пять упражнений с данными с использованием Spark SQL, включая:
   - анализ распределения базовой зарплаты `Base_Salary`;
   - анализ распределения зарплат по полу `Gender`;
   - связь базовой зарплаты и дополнительных выплат `Overtime_Pay`, `Longevity_Pay`;
   - влияние квартилей по уровню доплат на базовую зарплату;
   - зависимость базовой зарплаты от грейда `Grade`.
4. Построение графиков (гистограммы, столбчатые диаграммы, диаграммы рассеяния, линейные графики) для визуализации полученных результатов.
